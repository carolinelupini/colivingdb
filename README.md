# colivingDB

A free directory of 600+ nomad-style coliving spaces worldwide — every coliving on one map, filterable by country, room type, price, stay length, pop-up month, and vibe, with native-currency pricing and no login or email wall.

**Live at [colivingdb.com](https://colivingdb.com).**

## What's here

This repository is the deployed static site (GitHub Pages): the interactive homepage, pre-rendered country/city/listing pages for every entry, and the dataset that powers them (`data.js`).

- **Coverage:** permanent colivings, dated pop-up editions, and traveling programs across 80+ countries
- **Data:** compiled from public information and operator submissions; independently maintained — see [how we verify data](https://colivingdb.com/methodology/) and [how listings work](https://colivingdb.com/how-listings-work/)
- **Corrections:** spot something wrong? Use the feedback button on any page — accuracy is the whole point

Some outbound links are affiliate links — colivingDB may earn a commission at no extra cost to the visitor, which never affects what's listed or how it's ranked. Full disclosure on [how listings work](https://colivingdb.com/how-listings-work/).
