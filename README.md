# colivingDB

A fast, browse-first directory of nomad-style coliving spaces worldwide — search by region, country, room type, and vibe, see them all on one map, and click straight through to the operator. No booking gate, no questions asked.

Live at [colivingdb.com](https://colivingdb.com).

Currently a prototype with seed data that is **unverified** — prices are rough estimates, some pins are approximate, and dates change. Built to test the experience, not to book from.

Static single-page site: `index.html` + `data.js` (the dataset as `window.COLIVINGS`). Map by [Leaflet](https://leafletjs.com/) / [OpenStreetMap](https://www.openstreetmap.org/).
